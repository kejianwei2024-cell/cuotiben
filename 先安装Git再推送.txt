========================================
  先安装 Git，再运行「一键推送到你的仓库.bat」
========================================

报错「git 不是内部或外部命令」说明本机还没装 Git，按下面做即可。

一、下载并安装 Git
  1. 打开浏览器，访问：https://git-scm.com/download/win
  2. 会自动下载 64 位安装包（或按页面提示选你的系统）
  3. 双击下载好的安装包，一路「Next」
  4. 到「Adjusting your PATH environment」时，选默认的
     「Git from the command line and also from 3rd-party software」
  5. 后面都默认，点「Next」直到「Install」，装完点「Finish」

二、重新运行推送
  6. 关掉当前那个报错的黑色窗口
  7. 再双击运行：一键推送到你的仓库.bat
  8. 若弹出浏览器要登录 GitHub，用你的账号登录并授权
  9. 看到推送成功提示后，去 GitHub 仓库刷新，应能看到代码

三、然后按之前的「第二步」在网页上添加工作流文件，即可在 Actions 里打 APK。
